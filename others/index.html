# React-Native真机调试

  1. 使用xcode打开已经存在的项目中的.xcworkspace文件；

  2. 如图所示，选择你的设备（如果你的电脑与你的移动设备连接了，会在这里显示）

  3. 配置代码签名如果没有注册Apple developer accountopen in new window，就先去注册一个开发者账号，注册好之后，在project下选择targets中的与你工程名相同的文件 如图然后前往singning & capabilities，在Team选项中，选择你的开发者账号或者团队。如果遇到这样的提示：Failed to register bundle identifier.open in new window，可以通过修改Bundle Identity来解决：这个选项在General标签下面的Identify中。

  4. 编译并运行应用点击xcode右上角的Build and run按钮，或者使用快捷键Command⌘ + R来编译运行应用。这时需要保持你的设备为解锁状态，否则会运行失败。最后还需要让你的设备信任开发者的APP，前往设置>通用>设备管理，来信任开发者。这样就可以在自己的设备上运行开发应用了。
